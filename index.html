OAuth Redirect Helper

If you register this page URL as an LwA Redirect URI, it can display any ?code= (Authorization Code) or #access_token fragments sent by Amazon so you can copy them for server‑side exchange.

Detected parameters
    Copy the code/token into your backend exchange step and then remove it from the URL.

    Tip: Use a dedicated path like https://<username>.github.io/privacy/#oauth-callback or with query like ?source=amazon. Register the exact redirect in the LwA console.

    Quick Setup

    1. Replace the placeholders (company name, email, address) in this file.
    2. Commit as index.html to a public GitHub repository.
    3. Enable GitHub Pages → Deploy from branch → main / root.
    4. Use the resulting URL as your Privacy Policy URL in the Amazon Developer Console (LwA).
    5. (Optional) Also register this same URL as a Redirect URI for testing the consent flow.
